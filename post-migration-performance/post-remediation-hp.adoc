== 移行後 - オンラインFLI

LUN インポートを削除した後、LUN が `-space-reserve disabled`および `-space-allocation enabled`で設定されているかどうかを確認します。そうでない場合は、それに応じて修正してください。

SCSI UNMAP機能を正しく動作させるには、ベンダーのドキュメントを参照して、サポートされているソフトウェアバージョンを確認してください。この機能を有効にするには、Veritas VxVM + VxFSの組み合わせが必要です。

[cols="4"]
|===
| Filesystem | ブロックデバイス | LVM（FS） | Sanboot 


| ステップ1：マッピングされたすべてのHP-UXホストでストレージを再スキャンします | ステップ1：マッピングされたすべてのHP-UXホストでストレージを再スキャンします。 | ステップ1：マッピングされたすべてのHP-UXホストでストレージを再スキャンします | ステップ1：マッピングされたすべてのHP-UXホストでストレージを再スキャンします 
|===
.手順
. ストレージデバイスを特定します。
+
[source, cli]
----
ioscan -fnC disk
insf -e
sanlun lun show all
sanlun lun show all -pv
sanlun lun show -wwpn
----
. VGをインポートしてVGをオンラインにします。
+
[source, cli]
----
vgimport /dev/vg_name /dev/disk/disk_name
vgchange -a y /dev/vg_name
----
. ボリュームグループとファイルシステムを特定します。
+
[source, cli]
----
vgdisplay -v /dev/vg_name
vgdisplay /dev/vg_name
lvdisplay /dev/vg_name/lv_name
----
. ファイルシステムをマウントします。
+
[source, cli]
----
bdf
mount /dev/vg_name/vol_name /mount_point
----
