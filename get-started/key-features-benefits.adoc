= ONTAPのForeign LUN Importの主な機能とメリットについて学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP外部LUNインポート（FLI）は、外部（サードパーティ）アレイLUNからネイティブONTAP LUNにデータをコピーする組み込みのブロック移行機能で、オフラインとオンラインの両方の移行方式をサポートしています。FLIは、さまざまなサードパーティ製アレイからONTAPへSAN LUNを移行するように設計されており、さまざまなホストやアレイ間で計画、実行、検証を標準化するのに役立つ、一貫性のある移行方法論を提供します。



[[key-features]]
== 主な機能

* ビジネスのダウンタイム要件に合わせたオンラインおよびオフライン移行。
* ブロックレベルのOS非依存型移行では、データの移動にホストボリュームマネージャやOSユーティリティは使用されません。
* 異種SAN環境全体で動作し、さまざまなサードパーティアレイからSAN LUNをONTAPに移行するように設計されています。
* バックエンド トランスポートは、FCとiSCSIの両方のバックエンド接続に対応しており、柔軟性に優れていますが、サポートされる構成は相互運用性の検証結果によって異なります。
* 標準的なSANパスモデルを活用したネイティブマルチパスとロード バランシングのサポート。
* CLIによる制御と可視性により、再現性のある実行と運用監視が可能になります。




[[key-benefits]]
== 主なメリット

* ONTAP FLIはONTAPに含まれています。FLIベースのLUNインポートを実行するために、別途移行用アプライアンスは必要ありません。
* ホストへのアクセスが短時間の切り替え後に再開でき、インポート処理がバックグラウンドで継続されるオンラインワークフローを可能にすることで、ダウンタイムのリスクを軽減します。
* 異なるホストやアレイ間での計画、実行、検証を標準化するのに役立つ、一貫性のある移行方法論。
* 7 モードから ONTAP への移行など、複雑な移行シナリオをサポートします。移行ワークフローの一環として、アライメントやフォーマットに関する考慮事項に対処できます。
* 専門サービスは必須ではありませんが、強力にサポートされている運用モデルです。NetAppは、単純な移行を超えるあらゆる作業について、スコーピング、計画、トレーニングに関して専門サービスまたは認定パートナーの利用を推奨します。
