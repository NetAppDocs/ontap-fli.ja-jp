= Foreign LUN Import を使用した AFF への LUN のインポート
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 9.1 以降、 AFF はFLI をサポートします。FLI を使用すると、他のアレイから LUN をONTAPクラスタに直接インポートできます。

ONTAP 8.3.2 以降、 AFF は承認されたプロセス差異要求 (PVR) を使用して FLI をサポートできます。PVR を承認のために提出するには、 NetAppアカウント チームにお問い合わせください。承認されると、提出者 (通常はNetAppシステム エンジニア) は、FLI 機能を有効にするための手順が記載された承認レターを受け取ります。

8.3.2 より前のバージョンのONTAPソフトウェアでは、 AFFと同じクラスタ上の非AFF HA ペアに FLI インポートをステージングする必要があります。移行が完了したら、ボリュームまたは LUN 移動などの非中断操作 (NDO) を使用して、移行された LUN をAFFに移動できます。AFFクラスターに非AFFノードがない場合、これを容易にするためにスイング ギアを借りる可能性についてアカウント チームに相談してください。
