= ソースアレイの構成方法
:icons: font
:imagesdir: ../media/


[role="lead"]
イニシエータポートのホストエントリ（ LUN マスキング、ネットアップでは igroup と呼びます）を追加するには、ソースアレイのドキュメントを参照してください。この情報は、 Site Survey and Planning ワークシートのストレージグループセクションから取得できます。
