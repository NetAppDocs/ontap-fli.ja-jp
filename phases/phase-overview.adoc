= ONTAP FLI フェーズについて学ぶ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
このセクションでは、構造化された段階的なアプローチを使用して、Foreign LUN Import（FLI）移行（特にiSCSIバックエンドFLIに重点を置く）を実行するためのエンドツーエンドの方法論について説明します。検出、分析、計画と準備、実行、検証の各フェーズ全体にわたる目的、入力、検証チェックポイント、成果物を概説し、外部ストレージアレイからONTAPへの予測可能でサポート可能な低リスクの移行を確実にします。

このワークフローでは、事前のデータ収集と相互運用性の検証、ホストとファブリックの準備状況、明示的なiSCSIバックエンド設計の制約（IC LIFの使用、明示的なターゲットポータル/IQN、フルメッシュ接続、スケール制限）、およびアプリケーションの可用性とロールバックオプションを保護するための規律ある変更管理を重視しています。IMT主導のサポート検証、的を絞った修復計画、テスト移行、および厳格な移行後のクリーンアップを組み合わせることで、このアプローチは、FCベースおよびiSCSIベースのFLIシナリオの両方において、移行が正常に完了し、完全にサポートされ、十分に文書化され、運用上クリーンな移行後の状態を実現することを保証します。
